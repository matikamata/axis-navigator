Phala

fruition
Fruition consciousness arising immediately after path attainment — the mind 'falls into' Nibbāna. Four phala levels correspond to the four stages: Sotāpatti-phala, Sakadāgāmi-phala, Anāgāmi-phala, Arahatta-phala. Each phala permanently eradicates a set of fetters (saṃyojana).
